# ModernTech Solutions HR Management System

## Project Description

The ModernTech Solutions HR Management System is a front-end web application developed as a proof of concept to improve and centralise key human resources processes within the company. The application was built using Vue.js, Bootstrap, HTML, CSS, and JavaScript, and is designed to simulate how HR staff can manage employee records, payroll calculations, attendance tracking, and leave requests in one integrated system.

ModernTech Solutions is a growing software development company that specialises in custom healthcare software solutions. As the company expanded, its HR processes became increasingly difficult to manage through spreadsheets, email communication, shared calendars, and manually updated documents. This project demonstrates how a digital HR system can improve efficiency by centralising employee information and providing a more structured workflow for HR administration.

## Features

* Employee data management using dummy employee records
* Dashboard displaying HR summary statistics
* Payroll calculations and salary summaries
* Leave request management
* Attendance tracking
* Responsive user interface built with Bootstrap
* Client-side data handling using Vue.js and localStorage

## Technologies Used

* Vue.js
* Bootstrap 5
* JavaScript
* HTML5
* CSS3

## Setup Instructions

To run this project locally:

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open the terminal in the project folder.
4. Run the following command to install dependencies:

```bash
npm install
```

5. Start the development server:

```bash
npm run dev
```

6. Open the local Vite link shown in the terminal in your browser.

## Project Structure

The project is organised into components for each HR function, including dashboard, employee management, payroll, leave requests, and attendance. Dummy employee data is stored locally and used to simulate how a real HR system would operate on the client side.

## Key Functional Areas

### 1. Dashboard
Provides a summary of key HR information, including employee totals, monthly payroll, average salary, and department count, as well as a company overview.

### 2. Employee Management
Allows HR staff to view, add, search, and manage employee information using dummy data.

### 3. Payroll
Displays salary information and payroll-related calculations for employees.

### 4. Leave Requests
Allows leave requests to be recorded and managed through the interface.

### 5. Attendance
Tracks employee attendance records and provides a basic overview of attendance management.
